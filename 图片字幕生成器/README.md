# CaptionCraft - 图片字幕生成器

一个优雅的网页工具，为图片添加精美的多行字幕。采用复古胶片暗房美学设计，所有处理均在本地完成，保护隐私。

![复古胶片暗房风格](https://v2-three-xi.vercel.app/preview.png)

**在线体验**: [https://v2-three-xi.vercel.app](https://v2-three-xi.vercel.app)

## ✨ 功能特性

### 核心功能
- 🖼️ **图片上传** - 支持点击上传或拖拽上传（JPG/PNG/WebP，最大20MB）
- 📝 **多行字幕** - 支持最多50行字幕，每行独立显示
- 🎨 **实时预览** - 所有调整即时生效，所见即所得
- 💾 **一键导出** - 生成高清PNG图片，保留原图分辨率

### 字幕配置
| 参数 | 范围 | 默认值 |
|------|------|--------|
| 字幕高度 | 24-200px | 80px |
| 字体大小 | 12-120px | 40px |
| 字体颜色 | 任意颜色 | #FFFFFF |
| 轮廓颜色 | 任意颜色 | #000000 |
| 背景条颜色 | 任意颜色 | #000000 |
| 背景透明度 | 0-100% | 60% |
| 边框宽度 | 0-8px | 2px |
| 边框颜色 | 任意颜色 | #FFFFFF |
| 行间距 | 0-30px | 8px |
| 圆角 | 0-30px | 8px |

### 字体选择
- 微软雅黑
- 苹方
- 思源黑体
- 宋体
- 楷体
- Arial
- Georgia

### 水印功能
- ✏️ **自定义文字** - 默认"么么哒哒"，可自由修改
- 📍 **多种位置** - 顶部居中/左上/右上/底部居中/左下/右下/中心
- 🎨 **四种样式** - 毛玻璃/实心/描边/阴影
- 🔤 **大小调节** - 12-72px
- 👻 **透明度** - 20-100%

## 🎬 渲染效果

### 字幕布局
- 字幕显示在**图片下方**，不会遮挡原图内容
- 每行字幕有独立的背景条
- 背景条之间有清晰的分隔（切割感）
- 文字在背景条内水平和垂直居中

### 水印效果
- **毛玻璃**：半透明模糊背景 + 发光文字（默认）
- **实心**：深色背景 + 白字
- **描边**：轮廓描边 + 填充
- **阴影**：投影效果

## 🛠️ 技术栈

- **前端**: 原生 HTML5 + CSS3 + JavaScript
- **渲染**: HTML5 Canvas API
- **字体**: Google Fonts (Playfair Display, Source Sans 3)
- **部署**: Vercel

## 🔒 隐私保护

- 所有图片处理在浏览器本地完成
- 不上传任何图片到服务器
- 不存储用户数据

## 🚀 本地使用

```bash

git clone https://github.com/mark-1207/image-caption-generator.git

# 进入目录
cd image-caption-generator

# 直接用浏览器打开
open index.html
```

## 📝 使用流程

1. **上传图片** - 点击上传区或拖拽图片到页面
2. **调整字幕设置** - 在左侧面板配置字幕样式
3. **输入字幕内容** - 在文本框中输入多行字幕（每行一条）
4. **配置水印**（可选）- 调整水印文字、位置、样式
5. **生成图片** - 点击"生成字幕图片"按钮
6. **下载保存** - 点击"保存图片"下载PNG文件
